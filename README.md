## 多Agent代码审查与重构助手（适配小米MiMo计划）

## 项目简介
本项目是基于 Claude Code、Codex 与 DeepSeek 构建的自动化代码质量助手，解决个人开发者维护项目时，手动代码审查、技术债清理和重构效率低的痛点。

## 核心功能
- 多Agent闭环工作流：分析Agent → 重构Agent → 验证Agent
- 基于长上下文的存量代码问题识别
- 自动生成可执行的重构方案与代码修改建议
- 语法校验与单元测试闭环验证
